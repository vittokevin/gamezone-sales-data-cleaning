# Olist Logistics and Delivery Unit Performance Analysis

## Project Overview
This repository contains my university assessment analysing Olist’s Logistics and Delivery Unit using performance metrics, descriptive statistics, trend analysis, variance analysis, and strategic recommendations.

## Assessment Objectives
- Design a Metric Tree for the Logistics and Delivery Unit
- Identify a North Star metric
- Propose three input metrics
- Justify how the metrics help management assess business health
- Compare two groups using descriptive statistics
- Analyse trends over time
- Evaluate variance
- Recommend strategies for both groups

## Data Set
Olist Brazilian E-Commerce Public Dataset is a real-world dataset containing around 100,000 orders placed through Olist Store across multiple marketplaces in Brazil from 2016 to 2018. It includes data on orders, customers, products, payments, sellers, freight, delivery status, customer locations, and reviews, making it suitable for analysing logistics, delivery performance, and customer experience

## Tools and Technologies
Data Cleaning: Power Query
Exploratory Data Analysis: Excel
Data Visualisation: Tableau & Excel

## Methodology
Explain how the data was cleaned, grouped, analysed, and interpreted.
The data was cleaned by checking for missing values, inconsistent formats, and incorrect data types before analysis. No duplicate rows were found, which was a positive sign for data integrity. Since the missing values accounted for less than 1% of the dataset and my professor confirmed that removing them would not affect the integrity of the analysis, those records were deleted. In addition, the inconsistent formats and data type issues were recorded and corrected, with separate rows used to document the changes made for transparency.
